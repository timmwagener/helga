managing poses and transferring animation in Maya.
Comments, suggestions and bug reports are welcome.

======================================
Installing the Studio Library on Windows.
======================================

# After running the studioLibrary.exe, run the following two lines in the PYTHON script editor
import studioLibrary
studioLibrary.main()



======================================
Installing the Studio Library on OSX (video).
http://www.youtube.com/watch?v=2GyBAToYhSU&feature=channel&list=UL
======================================


1. Download & install PyQt4 for Maya 2012+ (OSX)
http://www.justinfx.com/2011/11/09/installing-pyqt4-for-maya-2012-osx/


2. Download the Studio Library
http://dl.dropbox.com/u/28655980/studioLibrary/studioLibrary.zip


3. Unzip and move the Studio Library folder to the maya scripts path...
*Replace {USERNAME} with your user name.
/Users/{USERNAME}/Library/Preferences/Autodesk/maya/scripts


4. Start Maya.


5. Run the following code in the Python script editor.
import studioLibrary
studioLibrary.main()
